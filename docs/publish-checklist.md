# Publish Checklist

Use this before publishing the template anywhere public.

## Privacy sweep

- [ ] No real email addresses
- [ ] No phone numbers
- [ ] No OAuth tokens
- [ ] No API keys
- [ ] No cookies / session exports
- [ ] No `.env` files
- [ ] No credential JSON files
- [ ] No private screenshots
- [ ] No raw chat logs
- [ ] No user-identifying notes in `USER.md`
- [ ] No curated private memories in `MEMORY.md`

## Repo hygiene

- [ ] `.gitignore` is present
- [ ] `LICENSE` is present
- [ ] `README.md` explains scope clearly
- [ ] branch is `main`
- [ ] commit history contains no private content

## Final review

- [ ] Read every tracked file once before push
- [ ] Confirm this is a template, not a sanitized production repo
- [ ] If in doubt, remove more
